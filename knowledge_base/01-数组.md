# 数组（Array）

数组是由相同类型元素组成的有序集合，在内存中连续存储，通过下标（索引）随机访问。

## 核心特性
- **连续存储**：元素物理地址相邻，CPU 缓存命中率高。
- **随机访问**：通过下标访问任意元素的时间复杂度为 O(1)。
- **固定大小**（静态数组）：创建后长度不可变；动态数组（如 C++ vector、Java ArrayList、Python list）可自动扩容。

## 时间复杂度
| 操作 | 复杂度 |
| --- | --- |
| 按下标访问 | O(1) |
| 头部插入/删除 | O(n) |
| 尾部插入/删除（摊还） | O(1) |
| 查找特定值 | O(n) |

## 扩容机制
动态数组容量满后，通常申请一块更大的内存（2 倍扩容），把旧元素逐个复制过去，摊还时间复杂度为 O(1)。

## 典型应用
一维数组用于线性序列；二维数组用于矩阵、棋盘、图像像素；前缀和、差分数组是高频优化技巧。

## 易错点
1. 数组越界（访问下标 ≥ 长度）。
2. 在遍历中删除元素导致下标错位（应从后向前删除或使用双指针）。
3. 扩容时浅拷贝导致多个引用共享同一块内存。
