# 树与二叉树（Tree & Binary Tree）

树是 n 个有限节点组成的具有层次关系的集合，有且仅有一个根节点，无环。

## 基本概念
- 节点的度、深度（depth）、高度（height）。
- 叶子节点（度为 0）、内部节点。
- 二叉树：每个节点最多两个子节点。

## 特殊二叉树
- **满二叉树**：每层节点都达到最大值。
- **完全二叉树**：除最后一层外全满，且最后一层从左到右连续——堆即用完全二叉树表示。
- **二叉搜索树（BST）**：左子树值 < 根 < 右子树值，中序遍历得到有序序列，查找/插入/删除平均 O(log n)，最坏退化为 O(n)。
- **平衡二叉树（AVL）/ 红黑树**：通过旋转保持平衡，保证 O(log n)。

## 遍历
- 深度优先：前序（根-左-右）、中序（左-根-右）、后序（左-右-根）。
- 广度优先：层序遍历（借助队列）。

## 时间复杂度（平衡时）
查找、插入、删除均为 O(log n)。

## 易错点
1. 删除 BST 节点时未正确处理"有两个子节点"的情况（需用前驱/后继替换）。
2. 递归遍历忘记终止条件导致栈溢出。
3. 混淆深度（从根到节点）与高度（从节点到最深叶子）。
