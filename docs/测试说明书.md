## 5. 已知限制
1. **mock 模式**：未配置星火密钥时输出占位内容；接入密钥后切换真实生成。
2. **RAG 检索**：当前为关键词重叠检索（零依赖）；可平滑升级为向量检索（见路线图）。
3. **语音**：视频/辅导为浏览器 Web Speech 占位配音，接入讯飞 TTS 后替换。
4. **持久化**：当前为内存态（Zustand），刷新后状态重置；可扩展 SQLite/Postgres。
