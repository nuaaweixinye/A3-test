# 并行启动指南

> 从零到能运行的最短路径。版本 v2.0

---

## 一、环境准备

前置：**Node.js ≥ 20.9**（推荐 22）、**Git**

```powershell
# 1. 克隆仓库
git clone git@github.com:nuaaweixinye/A3-test.git
cd A3-test

# 2. 安装依赖
npm install

# 3. 复制环境变量
cp .env.example .env
#   填入 SPARK_API_KEY（星火大模型）和 ChatDoc 凭证
#   未填则 mock 模式运行

# 4. 初始化数据库
npx prisma generate
npx prisma db push

# 5. 启动开发服务器
npm run dev          # http://localhost:3000
```

---

## 二、一键启动（发行包）

解压 `learning-multiagent.zip` → 双击 `启动系统.bat`

脚本自动完成：检测 Node.js → 初始化数据库 → 启动 dev server → 8秒后打开浏览器。

---

## 三、环境变量说明（`.env`）

```bash
# 讯飞星火大模型（必填，未填则 mock 模式）
SPARK_API_KEY=你的密钥
SPARK_BASE_URL=https://spark-api-open.xf-yun.com/v1

# 讯飞 ChatDoc 知识库（云端 RAG 检索）
SPARK_APP_ID=你的AppID
SPARK_API_SECRET=你的ApiSecret
SPARK_KB_REPO_ID=你的知识库ID

# JWT 认证
JWT_SECRET=随机字符串

# SQLite 数据库
DATABASE_URL="file:./dev.db"
```

## 四、常用命令

| 命令 | 作用 |
| --- | --- |
| `npm run dev` | 开发服务器（热更新） |
| `npm run build` | 生产构建 |
| `npm run lint` | ESLint 检查 |
| `npx tsc --noEmit` | 类型检查 |
| `npx prisma db push` | 同步数据库 schema |
| `npx prisma generate` | 生成 Prisma Client |
| `npx prisma studio` | 可视化查看数据库 |
